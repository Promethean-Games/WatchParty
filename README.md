## How to Play

Join or create a room, enter your name, pick an avatar

Share the room code so friends can join

(Optional) Host enables Team Play (Blue vs Red)

Switch teams by tapping your name (until roster is locked)

Choose an event list (pre-built or custom)

When an event happens on screen, tap it first to score

Host can pause during breaks and veto bad calls
